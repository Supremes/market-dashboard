📋 估值方法
市盈率 (PE)

{% if data.market == 'us' %}股票价格 / 每股收益。低于 15 偏便宜，20–25 合理，高于 30 偏贵。{% else %}股票价格 / 每股收益。港股整体 PE 偏低，恒指历史中位数约 10–12。{% endif %}

市净率 (PB)

股票价格 / 每股净资产。PB 越低说明相对于资产越便宜。低于 1 表示"破净"。

股息率

年度股息 / 股价。{% if data.market == 'us' %}标普500 历史均值约 1.5–2.0%。{% else %}港股高股息标的常见 4–6%，是重要的收益来源。{% endif %}

200 日均线偏离

当前价格相对 200 日均线的偏离程度。超过 +10% 可能过热，低于 -10% 可能超卖。

{% if data.market == 'us' %}
Shiller CAPE

10 年周期调整市盈率，用通胀调整后的平均收益计算，消除经济周期波动。历史中位数约 16–17。

风险溢价

盈利收益率 (1/PE) – 10 年期国债收益率。正值越大说明股票相对债券越有吸引力。

{% else %}
恒生指数 (^HSI)

香港股市最主要的指标，包含港股最大市值的 82 只成分股。是衡量港股整体表现的核心参考。

沪港深红利低波 (3110.HK)

追踪恒生高股息低波动指数的 ETF，覆盖沪港深三地高分红、低波动的优质公司。
