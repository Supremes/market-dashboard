📋 估值方法
{% if data.market == 'us' %}
市盈率 (PE)

股票价格 / 每股收益。低于 15 偏便宜，20–25 合理，高于 30 偏贵。

Shiller CAPE

10 年周期调整市盈率，消除经济周期波动。历史中位数约 16–17。

风险溢价

盈利收益率 (1/PE) – 10 年期国债收益率。正值越大股票越有吸引力。

{% elif data.market == 'hk' %}
恒生指数 (^HSI)

香港股市最主要指标，包含 82 只成分股。

沪港深红利低波 (3110.HK)

追踪恒生高股息低波动指数的 ETF。

{% elif data.market == 'cn' %}
沪深300

上海和深圳交易所中市值最大、流动性最好的 300 只股票。

中证500

扣除沪深300后，市值排名前500的股票。代表中盘股。

红利低波

高股息率+低波动率的股票组合。适合稳健型投资者。

RSI 指标

相对强弱指数。低于30超卖（可能反弹），高于70超买（可能回调）。

MACD

指数平滑移动平均线。金叉为看涨信号，死叉为看跌信号。

布林带

均线±2倍标准差通道。价格触下轨可能反弹，触上轨可能回落。

{% endif %}
市净率 (PB)

股票价格 / 每股净资产。PB 越低相对于资产越便宜。

股息率

年度股息 / 股价。高股息是长期投资的重要收益来源。
